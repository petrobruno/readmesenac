# Sobre o projeto

file:///C:/Users/Dell/Desktop/projeto_conecta_eletricista/frontend/frontend/public/login.html

A busca por profissionais de serviços domésticos e técnicos, como eletricistas e encanadores, é frequentemente dificultada pela falta de transparência e confiabilidade no mercado. Esta lacuna cria desafios significativos tanto para clientes, que lutam para encontrar profissionais qualificados e disponíveis, quanto para os próprios prestadores, que têm poucas ferramentas para construir sua reputação.
Este projeto visa preencher essa necessidade com o desenvolvimento de uma plataforma digital que conectará clientes e profissionais de forma rápida e segura. A solução tecnológica facilitará a solicitação de orçamentos e agendamentos, e a confiança será reforçada por um sistema robusto de avaliações e histórico de serviços. O objetivo é simplificar o processo de contratação para o cliente e oferecer aos profissionais um meio eficiente para expandir seus negócios, criando um ecossistema de serviço transparente e confiável.

# Como executar o projeto

## Back end
Pré-requisitos: Java 11

```bash
# clonar repositório
git clone [https://github.com/devsuperior/sds1-wmazoni](https://github.com/petrobruno/readmesenac/edit/main/README.md)]

# entrar na pasta do projeto back end
cd backend

# executar o projeto
./mvnw spring-boot:run
```

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone [https://github.com/devsuperior/sds1-wmazoni](https://github.com/petrobruno/readmesenac/edit/main/README.md)]

# entrar na pasta do projeto front end web
cd front-web

# instalar dependências
yarn install

# executar o projeto
yarn start
